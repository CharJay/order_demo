# 订单管理系统 Demo

一个轻量的订单管理 Demo 系统，实现订单的增删改查基础功能。

## 功能

- ✅ 订单列表（筛选、搜索、分页）
- ✅ 创建订单（表单 + 动态明细行 + 自动计算）
- ✅ 订单详情（只读 + 状态流转）
- ✅ 编辑订单（预填数据）
- ✅ 删除确认弹窗
- ✅ 状态流转（待处理 → 处理中 → 已完成/已取消）
- ✅ localStorage 数据持久化

## 技术栈

- React + TypeScript
- Vite
- localStorage 持久化

## 运行

```bash
npm install
npm run dev
```
